# Local Web Apps

ブラウザ単体で動作するローカルユーティリティ集。サーバー不要で、HTMLファイルを直接開いて使えます。

## 含まれるアプリ

### 1. URL Vault（高密度タブマネージャー）

URLをサムネイル付きカードとして管理・整理するタブマネージャーです。ブラウザのタブを大量に開きたくない場面で、URLを保存して視覚的に整理できます。

#### 特徴

- **3階層の整理構造** — ウィンドウ → グループ → アイテム でURLを体系化
- **サムネイル付きカード表示** — クリップボードから画像を貼り付け可能
- **ドラッグ＆ドロップで並び替え** — 直感的な操作で順序を変更
- **フィルタリング** — ウィンドウ・グループ単位で絞り込み表示
- **JSON形式のインポート/エクスポート** — データのバックアップと復元に対応
- **2行ペースト対応** — タイトルとURLの2行テキストを一度に貼り付け可能
- **IndexedDB使用** — ブラウザ内のローカルDBにデータを保存
- **ダークテーマUI** — 高密度表示に最適化されたレイアウト

#### 使い方

1. 「ウィンドウ枠作成」で大枠を作成（例：仕事、趣味）
2. 「グループ作成」でウィンドウ内にグループを作成（例：ドキュメント、SNS）
3. 「アイテム登録」でタイトル・URL・サムネイルを入力して保存
4. カードをクリックすると新しいタブでURLを開きます

---

### 2. WP Data Manager（系統・種牡馬データ管理）

競馬ゲーム（Winning Post）向けの系統・種牡馬データ管理ツールです。系統の滅亡防止や血統保護のプレイを補助します。

#### 特徴

- **系統データのCRUD管理** — 系統名・生年・種牡馬名を追加・編集・削除
- **ゲーム内年齢の自動計算** — ゲーム内年を設定すると年齢が自動表示
- **ドラッグ＆ドロップで行並び替え** — 表示順序のカスタマイズ
- **列ヘッダークリックでソート** — 順序・系統名・生年・種牡馬名でソート
- **CSV入出力** — CSV形式でのデータインポート・エクスポート
- **年代別馬リスト** — 史実馬データを内蔵し、ゲーム内年に基づいて表示
- **スケジュール管理** — チェックリスト形式の年間スケジュール（進行度を保存）
- **注記メモ** — 馬のマーク（☆◎○△）の定義など参考情報を表示
- **localStorage使用** — データはブラウザに自動保存

#### 内蔵史実馬データ

1963年〜2020年生まれの国内外の主要な競走馬・種牡馬データを搭載しています。ゲーム内年に応じて、直近に誕生する史実馬を確認できます。

#### 使い方

1. ゲーム内年を右上の入力欄に設定
2. CSVから読み込み、または手動で系統データを追加
3. 上部のトグルボタンでスケジュール・年代別リスト・メモを表示
4. スケジュールのチェックリストで作業進行を管理

## 共通仕様

| 項目 | 内容 |
|------|------|
| 動作環境 | モダンブラウザ（Chrome, Firefox, Edge, Safari） |
| サーバー | 不要（HTMLファイルを直接開く） |
| 外部通信 | なし（完全オフライン動作） |
| データ保存 | ブラウザ内（IndexedDB / localStorage） |

## ライセンス

このプロジェクトは個人利用目的のローカルツール集です。
